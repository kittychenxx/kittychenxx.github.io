重新认识img标签

img标签特点

  • 行内元素、可替换元素
  • 有src，可以设置图片的宽高；没有src，首先显示alt，没有alt再显示title，这里alt和title的内容高度还是我们设置的图片的高度，但是宽度就是替代内容alt、title的宽度
  • 默认的样式display:inline，当图片设置高度时，display:inline-block，才会显示正确的宽高

img标签间距问题

  1. 有宽高设置的<img> 默认display：inline-block，由于img标签是替换标签，所以可以根据存放的内容设置宽高，当父元素没有设置高度的时候，则用子元素的高度计算出高度给父元素，这时会出现3px的空隙问题
  2. img元素之间的存在空白，所有inline-block的元素之间都有空白

解决办法

  1. 给父元素添加高度0，font-size:0;
  2. img元素设置float属性
